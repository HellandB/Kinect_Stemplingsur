Vi ønsker å arbeide med en oppgave vi fikk anbefalt fra Bouvet.
Oppgaven går ut på å lage en fullgod erstatning til det nåværende
stemplingsuret. Der man bruker et id kort for å logge seg inn,
eller ut på arbeidsplassen. Målet med oppgaven vår er å finne ut
om det er mulig  å bruke  kinecten og dens funksjoner og å
implementere funksjonen.
